cd

cd (changes) the current/working (directory). One should familiarize himself with knowledge of directories on Linux.

Usage

  • cd [path]

Paths

There are shortcuts for specific paths one can use. Additional shortcuts for paths can usually be sourced for each new instance of the shell.

  • ./ the root folder
  • ../ the root of the root folder
  • ~ the current shell user's home directory, usually /home/user
