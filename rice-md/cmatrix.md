# cmatrix
![](../img/cmatrix.png)

cmatrix continously displays random text on a terminal screen.

## Usage
- cmatrix [options]

## Options
- **-c** Use Japanese characters. Requires appropriate fonts.
- **-C [color]** Set color.
- **-r** Rainbow mode
